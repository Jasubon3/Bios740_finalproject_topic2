# Main Results Summary

## ADKG
- Dev: NER micro F1 = 69.07, NER macro F1 = 57.74, RE micro F1 (w/o NEC) = 45.13, RE micro F1 (with NEC) = 40.78
- Test: NER micro F1 = 68.22, NER macro F1 = 61.89, RE micro F1 (w/o NEC) = 47.31, RE micro F1 (with NEC) = 42.50

## MDKG
- Dev: NER micro F1 = 80.99, NER macro F1 = 71.80, RE micro F1 (w/o NEC) = 55.25, RE micro F1 (with NEC) = 47.59
- Test: NER micro F1 = 81.08, NER macro F1 = 71.41, RE micro F1 (w/o NEC) = 56.70, RE micro F1 (with NEC) = 50.48

## Main takeaway
- MDKG performed better than ADKG on both entity extraction and relation extraction.
- Abbreviation_for was one of the strongest relation types.
- Associated_with and some broader semantic relations were harder.
